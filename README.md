# To-Do App (LocalStorage Version)

A simple and functional To-Do application built using **HTML, CSS, and JavaScript**.  
This project allows users to add, delete, edit, and mark tasks as completed.  
All tasks are stored in **localStorage**, so data persists even after refreshing the page.

---

## Features

- Add new tasks
- Delete tasks
- Mark tasks as completed
- Edit existing tasks
- Persistent storage using localStorage
- Task limit system (max 8 tasks)
- Simple and clean UI

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- LocalStorage API
